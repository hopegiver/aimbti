테스트용 API 키

아래 API 키들을 사용해서 테스트할 수 있습니다:

✅ 유효한 키 (localhost용)

test-key-12345

플랜: Free, 도메인: localhost

✅ 프리미엄 키 (example.com용)

demo-key-67890

플랜: Premium, 도메인: example.com

✅ 개발자 키 (모든 도메인)

dev-key-abcde

플랜: Unlimited, 도메인: * (모든 도메인 허용)

❌ 유효하지 않은 키

invalid-key-xxxxx

인증 실패 메시지를 확인할 수 있습니다.

사용 방법

<script type="module">
    import { AIMBTI } from './src/aimbti.js';

    const app = new AIMBTI('container-id', {
        apiKey: 'test-key-12345'
    });
    app.init();
</script>
